Detailed Description

Represents an image.

Note
Note that this class cannot be instantiated manually. Rather, it is created when calling Document::loadJPEGImageFromFile, Document::loadJPEGImageFromMemory, Document::loadPartialPNGImageFromFile, Document::loadPNGImageFromFile, Document::loadPNGImageFromMemory, Document::loadRawImageFromFile and Document::loadRawImageFromMemory.

Detailed Description

Represents a set of permissions for a pdf document.

The read permission is always included.

Permissions pdf::Permissions::operator- ( const Permissions &  other) const
noexcept

Creates new Permissions by taking the difference of two permission sets.

The read permissions will always be included in the result.

Note
This also works if subtracting permissions only included in other. Hence, Permissions::EDIT_CONTENT - Permissions::ALL works and is equal to Permissions::READ_ONLY.
